# Delegates

## Funcs

`Funcs<Result>`

`Funcs<T, Result>`

### 설명

Func를 여러개 추가할 수 있습니다.

Func의 결과값을 추가된 순서대로 계속 연산합니다.

`Result` 는 숫자만 가능합니다.

### 함수

`Add` 함수 추가

`Remove` 함수 제거
	
## Event

`Event<T>`

`Event<T1,T2>`

### 설명

실행할 함수를 설정하고 콜백들을 추가할 수 있는 클래스입니다.

### 함수

`Set` 함수 설정
 
`Add` 콜백 추가

`Remove` 콜백 제거

## EventFunc

`EventFunc<Result>`

`EventFunc<T,Result>`

`EventFunc<T1, T2, Result>`

### 설명

실행할 함수를 설정하고 결과를 반환하며 콜백들을 추가할 수 있는 클래스입니다.
	
`Result` 는 숫자만 가능합니다.
	
### 함수

`Set` 함수 설정
 
`Add` 콜백 추가

`Remove` 콜백 제거
